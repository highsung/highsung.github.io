常用的文件命令
  • pwd 显示当前目录
  • cd 切换目录
  • ls 显示目录内容
  • cat 显示文件内容
  • grep 查找文本中特定的字符串
    • 上面命令的含义：ps aux命令结果输入到一个管道中，再用grep后的参数过滤
    • | 代表管道，常常配合grep命令使用
    • grep后的参数，也可以正则表达式
  • touch 创建文件，或者更新文件创建时间
  • ln 创建链接

    ln命令用来为文件创件链接，连接类型分为硬链接和符号链接两种，默认的链接类型是硬连接。如果要创建符号链接必须使用”-s”选项

    • 格式：ln -s 源文件 目标文件
  • mv 移动文件或者修改文件名
  • cp 复制文件
    • 复制目录加-r选项
  • rm 删除文件
    • 删除目录加-r选项
  • mkdir 创建目录
  • chmod 修改文件权限
    • 7（111二进制）代表对文件有rwx的权限，6（110二进制）代表对文件有rw权限，没有x权限。
    • +x，+w代表增加x，w权限，-x，-w代表取消x，w权限。
  • find 查找文件
  • tar 压缩和解压
用户管理
  • useradd 添加用户
  • userdel 删除用户
  • usermod 修改账号
  • passwd 修改用户密码
  • groupadd 添加用户组
  • groupdel 删除用户组
  • newgrp 切换用户组
  • sudo 赋予普通用户root权限
网络配置
  • ifconfig
软件安装
  • yum
进程管理
  • kill -9 17788
  • netstat -tln | grep 8080
  • lsof -i :8080
  • ps aux|grep java 查看java进程
  • ps -ef
